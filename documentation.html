Tentang Kami

JstMStudio ialah sebuah studio kecil yang diasaskan oleh Jst Tan. JstMStudio memberi tumpuan kepada banyak perkara, daripada muzik, penulisan cerita, pembangunan laman web dan banyak lagi. JstMStudio ialah studio indie terpencil kecil yang memfokuskan pada banyak perkara untuk membina masa depan yang lebih baik.

API

JstMStudio menyediakan banyak perkhidmatan dan ciri dalam perisian platform kami, Platform Aplikasi JstMStudio. API JstMStudio ialah Antara Muka Pengaturcaraan Aplikasi untuk Platform Aplikasi JstMStudio yang menyediakan sokongan untuk semua pembangun untuk melaksanakan perkhidmatan seperti dalam Platform Aplikasi JstMStudio ke dalam tapak web mereka tanpa membangunkan ciri itu sendiri secara percuma tanpa sebarang bayaran. Ia dibina untuk digunakan oleh semua orang dan bukan untuk mendapatkan wang daripada pengguna seperti pembangun dan pemilik tapak web.

API JstMStudio ialah API sumber terbuka ( Antara Muka Pengaturcaraan Aplikasi ). Semua orang boleh melihat, membaca dan mengkaji semua kod sumber API JstMStudio pada repositori Github rasmi kami. API JstMStudio dibina menggunakan HTML, CSS dan JavaScript vanila tidak seperti mana-mana API tradisional juga dan dilesenkan di bawah AGPL 3.0.
